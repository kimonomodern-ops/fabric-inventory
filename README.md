KIMONO MODERN 社内向け生地在庫管理ツールです。

## セットアップ手順

### Step 1：GitHub Pages でホスティング

1. GitHub.com でリポジトリ `fabric-inventory` を **Public** で作成
2. `index.html` をアップロード
3. Settings → Pages → Source: `main` / `/(root)` → Save
4. 数分後に `https://<ユーザー名>.github.io/fabric-inventory/` が発行される

### Step 2：Google Sheets + GAS のセットアップ

1. Google スプレッドシートを新規作成
2. URLから **スプレッドシートID** をコピー（`/d/` と `/edit` の間の文字列）
3. メニュー「拡張機能」→「Apps Script」を開く
4. `fabric_inventory_GAS.js` の内容を貼り付け
5. 3行目の `YOUR_SPREADSHEET_ID_HERE` を実際のIDに書き換え
6. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」
7. アクセス権：「全員」→ デプロイ → **WebアプリURL** をコピー

### Step 3：アプリとGASを接続

1. GitHub Pages の URL をブラウザで開く
2. 上部の GAS URL 入力欄に Step 2 の URL を貼り付け
3. 「接続」ボタンをクリック → 緑の「GAS同期中」が表示されれば完了

### Step 4：社員への共有

GitHub Pages の URL を LINE や Slack で共有するだけ。ログイン不要。

## 機能一覧

- **在庫一覧**：品番・品名・色・仕入先でフリーワード検索、カテゴリ・単位フィルター、統計サマリー
- **インライン入出庫**：一覧から直接数量を入力して入庫↓ / 出庫↑
- **在庫追加・編集**：モーダルフォームで全フィールドを入力
- **入出庫詳細**：種別（入庫・出庫・棚卸調整）・担当者・備考を記録
- **出荷処理**：品目選択 → 数量・担当者・送り先を入力して「出荷」記録
- **入出庫履歴**：種別タブフィルター・品番キーワード絞り込み・履歴クリア
- **一括インポート**：CSV/TSV ドラッグ&ドロップ、コピペ対応、列マッピング
- **CSV エクスポート**：UTF-8 BOM付き、ファイル名 `生地在庫_YYYY-MM-DD.csv`

## 注意事項

- **認証なし**：URL を知っていれば誰でも操作できます（パスワード保護は別途対応）
- **GAS無料枠**：1日あたり実行時間6時間・呼び出し20,000回
- **写真**：外部サービス（Google Drive等）に画像をアップしてURLを取得してください
- **オフライン**：GAS連携時はオフライン不可（GAS未接続時はブラウザローカル保存で動作）
- **同時編集**：排他制御なし。複数人が同時操作した場合、最後の操作が反映されます
